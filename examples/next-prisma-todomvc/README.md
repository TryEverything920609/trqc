## TodoMVC

- [TodoMVC](https://todomvc.com/) implemented with tRPC + [Prisma](https://prisma.io)

### Setup

- Get postgres running
- `CREATE DATABASE todo`
- Possibly tweak `.env` with your pg user


### Useful commands

```bash
yarn dx # runs prisma studio + next
```
